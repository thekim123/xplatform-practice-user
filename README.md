# xplatform-practice-user
xplatform 유저 기능 구현 프로젝트 백앤드(Springboot)

xplatform 교육자료에 있는 것을 스프링부트, MariaDB를 이용하여 백앤드를 구성했습니다.

xplatform 라이센스나 api.jar는 올라와있지 않습니다.
